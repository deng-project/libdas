\hypertarget{index_intro_sec}{}\doxysection{Introduction}\label{index_intro_sec}
\mbox{\hyperlink{namespaceLibdas}{Libdas}} is a small library that features custom file format reading and writing features, collision detection and parsers for third party file formats. DAS is an acronym that stands for DENG asset format. The format features data buffers, texture image buffers, Huffman compression, scene hierarchies and animations.\hypertarget{index_build_sec}{}\doxysection{Building}\label{index_build_sec}
\mbox{\hyperlink{namespaceLibdas}{Libdas}} can be built using premake5 with following commands\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$ premake5 gmake2}

\end{DoxyCode}
 and 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$ premake5 vs2019}

\end{DoxyCode}
 when using GNU/\+Make and Visual Studio respectively.
